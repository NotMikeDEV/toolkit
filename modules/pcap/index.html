Cloud Shark Wrapper

Save as /sbin/cshark and chmod +x.

Copy/Paste (as root): 

wget -O /sbin/cshark http://tools.as206671.uk/pcap/cshark; chmod +x /sbin/cshark

To use this server with the OpenWRT CloudShark client, use server URL http://tools.as206671.uk:8080/pcap/ and any token.

These tools are provided for the sole purpose of showing how easy remote packet capture can be, not as an actual service. Files are regularly deleted.
